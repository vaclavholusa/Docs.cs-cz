<span data-ttu-id="5b38e-112">Ve většině případů navrhnout strategie nasazení pro databázi, první věc, kterou je potřeba zvážit při jaká data chcete nasadit.</span><span class="sxs-lookup"><span data-stu-id="5b38e-112">In most cases, when you devise a deployment strategy for a database, the first thing you need to consider is what data you want to deploy.</span></span> <span data-ttu-id="5b38e-113">V prostředí vývoj nebo testování můžete chtít nasadit dat účtu uživatele k usnadnění rychlé a jednoduché testování.</span><span class="sxs-lookup"><span data-stu-id="5b38e-113">In a development or test environment, you might want to deploy user account data to facilitate quick and easy testing.</span></span> <span data-ttu-id="5b38e-114">V testovací nebo produkční prostředí je velmi pravděpodobné, že chcete nasadit dat účtu uživatele.</span><span class="sxs-lookup"><span data-stu-id="5b38e-114">In a staging or production environment, it's very unlikely that you'd want to deploy user account data.</span></span>
