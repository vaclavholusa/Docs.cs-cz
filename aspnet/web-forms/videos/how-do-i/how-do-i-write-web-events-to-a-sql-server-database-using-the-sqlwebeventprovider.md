<a name="how-do-i-write-web-events-to-a-sql-server-database-using-the-sqlwebeventprovider"></a>[Postup:] Zápis webových událostí do databáze systému SQL Server pomocí SqlWebEventProvider
====================
podle [Chris pixelů na](https://twitter.com/chrispels)

V toto video pixelů na Chris ukazuje, jak přihlášení chyby webové stránky k databázi SQL serveru pomocí SqlWebEventProvider monitorování stavu v ASP.NET. Nejdřív zjistěte role zprostředkovatele a události v monitorování stavu technologie ASP.NET. Dále uvidíte, jak nakonfigurovat databázi systému SQL Server pomocí nezbytných objektů pro sledování události pomocí aspnet záznam stavu\_regiis nástroj, stejné nástroje použít ke konfiguraci členství technologie ASP.NET. Pak zjistěte, jak nakonfigurovat sledování stavu v souboru web.config pro záznam událostí v webu ASP.NET na nově vytvořenou databázi systému SQL Server. V rámci této konfigurace se podívejte, jak v kořenovém souboru web.config v rozhraní .NET Framework 2.0 poskytovatelů a definované události, které můžete využít při konfiguraci monitorování stavu. Tyto základní informace jsou základní na kterém je možné vytvořit vlastní události, které protokolovat konkrétní informace do databáze SQL serveru na webu technologie ASP.NET.
